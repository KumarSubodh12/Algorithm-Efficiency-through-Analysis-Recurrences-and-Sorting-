## 🎯 Overview
This assignment analyzes algorithm efficiency using empirical time
measurement and theoretical time complexity. Various algorithms are
implemented in Python and compared using graphical visualization.
The results validate theoretical complexity with practical execution time.

---

### Task 1 – Algorithm Growth Analysis
Comparison of:
- O(1)
- O(n)
- O(n²)
- O(log n)

### Task 2 – Searching Algorithms
Comparison of:
- Linear Search
- Binary Search
(Best, Average, Worst cases)

### Task 3 – Recursive Algorithms
- Factorial
- Fibonacci

### Task 4 – Recurrence Relations
- T(n) = T(n/2) + n
- T(n) = 2T(n/2) + n
